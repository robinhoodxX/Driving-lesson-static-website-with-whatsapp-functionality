Driving Lesson – GreenPass is a static website designed to streamline the process of managing driving lessons, learners, and instructors efficiently. 
The system provides a digital platform for driving schools, students, and instructors to connect, schedule lessons, track progress, and maintain learner records.
GreenPass simplifies how driving schools operate by replacing manual paperwork with an interactive online system using Whatsapp. 
Students can register, book driving sessions, and while instructors can manage lesson schedules, update feedback, and monitor learner performance.

This project focuses on modern web technologies to deliver a responsive, user-friendly, and secure experience.


User Roles – Separate forum for Students, Instructors, and Admins.
Lesson Booking System – Students can book driving sessions with instructors directly and rest instructor will caterorize it.
Instructor Management – Instructors can manage schedules, student feedback, and progress reports.
Notifications & Alerts – Automatic notifications for bookings, and reminders.
Responsive UI – Built with React.js, Material UI, and Next.js for a seamless multi-device experience.

Frontend: HTML5, CSS, Javascript
Language: JavaScript
Version Control: Git & GitHub
